# Change Log

All notable changes to the "REDAXO DDEV Manager" extension will be documented in this file.

## [1.0.0] - 2024-12-19

### Added
- Initial release of REDAXO DDEV Manager
- DDEV project lifecycle management (create, start, stop, restart, delete)
- Automatic REDAXO installation with version selection (5.15+)
- Modern webview-based dashboard UI
- TreeView integration for project status visualization
- Database import/export functionality
- Built-in development tools integration (Mailhog, phpMyAdmin, Xdebug)
- SSL certificate support via DDEV
- Project sharing capabilities with ngrok integration
- Multi-PHP version support (7.4-8.4)
- MySQL and MariaDB database options
- Terminal integration for container access
- Project URL management and browser launching

### Features
- **Beautiful Dashboard**: Modern, responsive webview interface for project management
- **One-Click Operations**: Start, stop, restart projects with single clicks
- **Automatic Setup**: Complete REDAXO installation and configuration
- **Development Ready**: Pre-configured with essential development tools
- **Flexible Configuration**: Support for multiple PHP and database versions
- **VS Code Integration**: Native integration with VS Code's sidebar and commands

### Requirements
- DDEV installed and configured
- Docker Desktop or compatible container runtime
- VS Code 1.74.0 or higher
